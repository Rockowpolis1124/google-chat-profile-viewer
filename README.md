# Usage

1. このリポジトリをダウンロード
2. Chromeを開き、拡張機能の画面でデベロッパーモードをオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. ダウンロードフォルダ内のsrcフォルダを選択する
5. 村のチャットを開く

将来的には限定公開でChromeのExtentionとしてwebで配布したい。
